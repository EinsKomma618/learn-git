learn-git
=========

A git repository to learn how to use Github

Check out the [wiki](https://github.com/Pitt-CSC/learn-git/wiki) for instructions!

## Slideshow

Want to run the slideshow? It's easy!

1. Install `reveal-md`

    ```bash
    $ cd path/to/this/repo
    $ npm install
    ```

2. Start the slideshow

    ```bash
    $ npm start
    ```
